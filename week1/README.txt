---Flight Trajectory Simulator

Este proyecto simula la trayectoria de un proyectil usando ecuaciones basicas de movimiento.

Tecnologicas usadas:
- Python
- Numpy
- Matplotlib

El programa calcula:
- Trayectoria completa
- Altura maxima alcanzada
- Altura minima alcanzada 
- Puntos clave de la trayectoria

El programa limite:
- Velocidades negativas
- Angulos negativos y mayores a 90°

---Flight Data Analysis

Este proyecto, toma valores no reales de un vuelo, con parametros, tales como:
1.  Altitud
2.  Velocidad
3.  Consumo combustible

De forma tal que los pogamos en un data frame, con un tiempo de 0<time<100.
Luego podamos visualizar en columnas y filas, para guardarlo en un csv.

El programa utiliza valores no reales a un vuelo, solo es a modo de practica.
